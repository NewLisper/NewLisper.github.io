fixpoint

对于一个函数f，其不动点是定义域中的元素x，满足f(x) = x。
例如f(x) ＝ x^2，0和1就是其不动点。

problem to solve

在lambda演算中不支持lambda项的命名，然而递归函数需要引用自身才能定义，那么需要一种方法使用匿名函数来实现递归函数。
用Scheme语言来描述这个问题（Scheme基本可看作lambda演算的扩展)：
对于递归函数fac


 fac = (lambda(n)
         (if (= n 0) 1
           (* n (fac (- n 1)))))
需消去等式右边的fac，改写以上代码为： 

 fac = (lambda(f)
        (lambda(n)
          (if (= n 0) 1
            (* n (f (- n 1)))))) fac
定义等式右边非fac部分为函数H 

 fac = H fac
则问题转化为求函数H的不动点 ##Y combinator Y combinator是由Haskell Curry发现的计算lambda演算中函数不动点的高阶函数。 

Y = (lambda(f) 
     ( (lambda(x) f (x x))
       (lambda(x) f (x x)) ))
对于任意函数h 

Y h = (lambda(f) 
     ( (lambda(x) f (x x))
       (lambda(x) f (x x)) )) h

    = ( (lambda(x) h (x x)) 
        (lambda(x) h (x x)) )
    = h ( (lambda(x) h (x x))
          (lambda(x) h (x x)) )
    = h (Y h)
满足定义。 ##verification 

fac 0 = Y H 0
      = H (Y H) 0
      = (lambda(f)
         (lambda(n)
          (if (= n 0) 1
            (* n (f (- n 1)))))) (Y H) 0
      = (lambda(n)
          (if (= n 0) 1
            (* n ((Y H) (- n 1))))) 0
      = 1

fac 1 = Y H 1
      = H (Y H) 1
      = (lambda(f)
         (lambda(n)
          (if (= n 0) 1
            (* n (f (- n 1)))))) (Y H) 1
      = (lambda(n)
          (if (= n 0) 1
            (* n ((Y H) (- n 1))))) 1
      = (* 1 ((Y H) 0))
      = (* 1 1)
      = 1
